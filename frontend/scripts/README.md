# Scripts

Thư mục này chứa các script tiện ích cho dự án VietSign.

## Utility Scripts

### `convertExcel.js`

Script chuyển đổi dữ liệu từ file Excel sang format JavaScript/TypeScript.

**Chạy script:**

```bash
node scripts/convertExcel.js
```

**Lưu ý:**

- Đảm bảo đã cài đặt dependencies: `npm install xlsx`
- Đặt file Excel cần convert vào thư mục gốc hoặc chỉ định đường dẫn trong script
